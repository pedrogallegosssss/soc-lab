SOC Lab — Ubuntu Attack Mapping Mini Project

Objective:
Practice identifying attacker behavior and mapping it to MITRE ATT&CK.

What I did:
- Simulated basic system discovery (whoami, id, hostname)
- Simulated command execution
- Practiced mapping actions to ATT&CK techniques

Key Mappings:
- Discovery → T1082 (System Information Discovery)
- Execution → T1059 (Command and Scripting Interpreter)

Key Takeaway:
Focus on attacker intent, not just commands.
